# DriverKnowledgeTest

Project skeleton for a driving knowledge test practice app using a layered architecture:

- `src/DriverKnowledgeTest.Web` - ASP.NET Core Razor Pages UI
- `src/DriverKnowledgeTest.Application` - use cases and service contracts
- `src/DriverKnowledgeTest.Domain` - entities and domain rules
- `src/DriverKnowledgeTest.Infrastructure` - data providers and wiring

## Next steps

1. Install .NET SDK (planned for .NET 10 preview).
2. Run `dotnet restore`.
3. Run `dotnet run --project src/DriverKnowledgeTest.Web`.
